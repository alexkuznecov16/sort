Kā tas darbojas?

1. Ievads

Šis projekts nepieciešams tam, lai lietotājs varētu sakārtot skaitļus vai burtus nepieciešamajā secībā. Tālāk esošajā laukā ir sniegts JavaScript kods ar detalizētiem šī projekta skaidrojumiem.

2. JavaScript kods

Projektā tiek izmantotas četras galvenās funkcijas:
  1. pārvaldnieka funkcija 'menedzFunk()' datu iegūšanai no lietotāja, visu pārējo funkciju izsaukšanai un gala rezultātu izdrukāšanai (šī funkcija tiek aktivizēta, noklikšķinot uz pogas 'Aprēķināt');
  2. funkcija vērtību testēšanai 'parbaudFunk(ievaditi skaitli/burti, vertibas tips, kartosanas tips)', ko ievadījis un izvelejis lietotājs;
  3. galvenā funkcija 'galvenaFunk(rezultats)', lai atrisinātu doto projektu;
  4. funkcija, lai notīrītu visas lauku vērtības 'dzestFunk()' pirms jauno vērtību palaišanas (šī funkcija tiek aktivizēta, noklikšķinot uz pogas 'Dzēst visu');
Šī projekta galvenās funkcijas JavaScript kods ir parādīts laukā zemāk: 
oninput="this.value = this.value.replace(/[^0-9]/g, '');"
// - ir funkcija, kur onInput (oninput = kad input bus aktivs) this.value (sis inputa value) .replace(/[^0-9]/g, ''); (mainis visu un drikst izmantot tikai skaitlus)
Noklikšķiniet uz šīs ikonas , lai skatītu šī JavaScript koda blokshēmu.
